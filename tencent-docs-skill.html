腾讯文档编辑 Skill

让 Claude Code 直接创建和编辑腾讯文档：智能文档、表格、PPT、思维导图，全类型支持

🚀 安装方式

在终端复制下面两条命令，逐条执行即可。需要先装 SkillHub CLI，再通过它安装 tencent-docs skill：

第一步：安装 SkillHub CLI

bash curl -fsSL https://skillhub.cn/install/install.sh | bash

第二步：安装 tencent-docs skill

bash skillhub install tencent-docs
📋

装好之后能干什么？

装好之后，Claude Code 就能直接操作腾讯文档了。不需要打开网页手动编辑，直接在对话里说：

"帮我创建一个智能文档，标题是xxx"
自动创建排版美观的在线文档，支持标题、列表、表格、代码块、分栏等
"帮我把这段内容保存到腾讯文档"
直接把对话中生成的内容沉淀到云端文档
"帮我创建一个 Excel 表格"
创建在线表格，支持公式、筛选、合并单元格等
"帮我做一个 PPT"
自动生成幻灯片演示文稿
"帮我画一个思维导图 / 流程图"
创建思维导图或流程图，可视化展示知识结构
"帮我修改那个文档里的xxx"
读取已有文档内容，定位到具体位置进行修改
📄

支持的文档类型

一个 Skill 覆盖了腾讯文档的全部品类：

智能文档 排版最美观，适合写文章、报告、笔记
Word 文档 传统格式，支持精细排版
Excel / 智能表格 数据处理、公式计算、多视图
PPT 幻灯片 AI 生成演示文稿
思维导图 层次化知识整理
流程图 流程架构可视化
💡

实际使用案例

比如你想把和 Claude Code 的对话内容整理成文档保存：

你只需要说
帮我把今天聊的网站搭建过程整理成一篇智能文档，标题叫「个人网站搭建与维护教程」，保存到腾讯文档

Claude Code 会自动创建文档、排版、插入目录和章节，并返回文档链接。你直接在浏览器打开就能查看和分享。

注意事项

首次使用需要授权

第一次调用腾讯文档时，会提示你登录授权。按提示扫码或点击链接登录即可。

内容太长时的技巧

如果文章内容很多（超过两三千字），Claude Code 可能会遇到命令行长度限制。这时候告诉它「分段追加到文档末尾」，它会自动拆开处理。

需要腾讯文档账号

生成的文档保存在你的腾讯文档账号下（docs.qq.com），需要有腾讯文档账号（微信/QQ 登录即可）。
