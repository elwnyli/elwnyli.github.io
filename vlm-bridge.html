VLM Bridge：让 DeepSeek / Kimi 看懂图片

纯文本模型本来看不懂图片，装了这个 Skill 之后，截图拖进去 AI 就能看图说话了

🚀 最快安装方式（推荐）

下载下面的 .md 文件，先准备好一个视觉模型 API Key（推荐 ModelScope，免费注册即得），然后把文件发给 Claude Code 说：

帮我按照这份文档配置 VLM Bridge，我的 API Key 是 ms-xxxxx

Claude Code 会自动完成全部配置。配完后必须重启 Claude Code。
